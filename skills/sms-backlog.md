---
name: sms-backlog
description: Move the current branch (or a named branch) to the backlog state in the sms tree, hiding it from default listings. Use when the user wants to set work aside without marking it merged. Runs `sms backlog`.
---

Run via the Bash tool:

  sms backlog [branch]

If the user named a branch, pass it; otherwise default (uses current branch).
