---
name: sms-tree
description: Show the branch tree managed by sms (Session Management System). Use when the user asks to see, browse, or visualize the branch/session tree. Runs `sms tree`.
---

Run `sms tree` via the Bash tool and print the output verbatim.

If the user passes `--all`, include `--all` in the invocation.

The output is a text tree of branches; current branch is marked with ★, session counts in parens, branch state in brackets when not active.
